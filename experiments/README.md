# RepMint Validation Experiments

This folder contains the assignment-ready and launch-ready validation pack for RepMint.

## What To Run

Run these three experiments in order:

1. Landing page smoke test: validates whether the promise earns action.
2. Goal-based training survey: validates which customer segment, goal, and feature bundle matters most.
3. Concierge camera-coaching pilot: validates whether the coaching experience is useful before building the full app.

## Files

- `VALIDATION_PLAN.md`: full experiment design, hypotheses, metrics, success criteria, and timeline.
- `SURVEY.md`: ready-to-build survey questions and scoring guide.
- `CONCIERGE_PILOT.md`: participant flow, moderator script, privacy language, and post-session questions.
- `LAUNCH_CHECKLIST.md`: launch tasks, tracking setup, channels, and submission checklist.
- `RESULTS_TRACKER.csv`: spreadsheet-ready template for recording experiment results.
